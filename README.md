# Tutorial-Instal-Android-Studio
Repository ini berisi tutorial langkah demi langkah untuk melakukan instalasi Android Studio.  Di dalamnya terdapat panduan lengkap mulai dari proses download, instalasi, hingga konfigurasi awal  yang dibutuhkan agar Android Studio siap digunakan untuk pengembangan aplikasi Android.


# Langkah 1: Membuka Chrome
Buka aplikasi Google Chrome di laptop atau PC kamu. Chrome akan digunakan untuk mengakses situs resmi Android Studio dan mendownload file instalasi yang dibutuhkan.

# Langkah 2: Mencari Android Studio
Pada kolom pencarian Google, ketik kata "Android Studio" lalu tekan Enter. Hasil pencarian akan menampilkan situs resmi Android Studio dari developer Android (developer.android.com).

# Langkah 4: Scroll Kebawah Halaman
Setelah masuk ke halaman resmi Android Studio, lakukan scroll ke bawah sampai menemukan tombol "Download Android Studio".  Tombol ini biasanya tersedia sesuai dengan sistem operasi yang kamu gunakan, misalnya Windows, macOS, atau Linux.

# Langkah 5: Pilih Installer Windows Yang Direkomendasikan Sesuai Sistem PC/Laptop
Pada bagian download, pilih file installer sesuai dengan sistem operasi yang kamu gunakan:  

- Jika menggunakan *Windows*, pilih file dengan format .exe yang bertuliskan "Direkomendasikan".  
- Jika menggunakan *MacBook (macOS)*, pilih file dengan format .dmg sesuai dengan chip perangkat kamu (Intel atau Apple Silicon/ARM).  
- Jika menggunakan *Linux*, pilih file dengan format .tar.gz yang tersedia.  
Pastikan memilih installer yang sesuai agar proses instalasi berjalan lancar.

 # Langkah 6: Mulai Download dan Tunggu Proses Selesai

Setelah memilih installer sesuai sistem operasi, klik link download tersebut. Proses pengunduhan akan dimulai secara otomatis. Tunggu hingga file installer selesai terunduh dengan sempurna sebelum melanjutkan ke proses instalasi. Ukuran file biasanya sekitar 1,5 GB, jadi pastikan koneksi internet stabil.

# Langkah 7: Membuka File Installer dan Memulai Instalasi
Setelah file installer berhasil diunduh, buka file tersebut dengan cara klik dua kali. Jika muncul notifikasi dari Windows/macOS/Linux untuk konfirmasi instalasi, pilih "Yes" atau "Allow". Proses instalasi Android Studio akan dimulai melalui jendela setup wizard.

# Langkah 8: Ikuti Setup Wizard
Setelah jendela instalasi terbuka, klik tombol "Next" untuk melanjutkan. Pada bagian pilihan komponen, biarkan default (Android Studio dan Android Virtual Device tercentang), lalu klik "Next".

# Langkah 9: Pilih Lokasi Instalasi
Tentukan folder lokasi instalasi Android Studio. Biasanya secara otomatis diarahkan ke folder Program Files (Windows) atau Applications (macOS). Disarankan biarkan default, lalu klik "Next".

# Langkah 10: Mulai Instalasi
Klik tombol "Install" untuk memulai proses instalasi. Tunggu beberapa menit hingga proses selesai. Durasi instalasi bisa berbeda tergantung performa laptop/PC kamu.

# Langkah 11: Selesai dan Jalankan Android Studio
Setelah instalasi selesai, klik tombol "Finish". Jika muncul pilihan untuk langsung membuka Android Studio, centang opsi tersebut lalu klik "Finish".  Android Studio akan terbuka untuk pertama kalinya.

# Langkah 12: Setup Awal Android Studio
Saat pertama kali dijalankan, Android Studio akan meminta konfigurasi awal. Pilih "Standard" untuk pengaturan default, lalu klik "Next". Pilih tema (Light/Dark), lalu klik "Next". Android Studio akan otomatis mendownload komponen tambahan (SDK, emulator, dll). Tunggu proses selesai, lalu klik "Finish".

# Langkah 13: Android Studio Siap Digunakan
Setelah semua konfigurasi selesai, tampilan utama Android Studio akan muncul. Sekarang kamu sudah siap membuat project Android pertamamu.
